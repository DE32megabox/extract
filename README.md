# movie_extract/movie_e.py
해당 년월의 영화 데이터 추출 모듈

## 로컬 개발 환경 설치

Extract 패키지를 로컬에서 개발하거나 수정하려면 다음 단계를 따라야합니다.

1. 레포지토리를 클론합니다.
```bash
$ git clone git@github.com:DE32megabox/extract.git
```
2. 클론한 디렉토리로 이동합니다.
```bash
$ cd megabox/code/movie_extract
```
3. 가상 환경을 활성화하고 패키지를 설치합니다.
```bash
$ pdm init
$ source .venv/bin/activate
$ pdm install
```
